Chapter 4 Terminology
Terms and Definitions
Network Classifications
    LAN: local area network (LAN); consists of collection of computers in a single building or building complex.
    MAN: metropolitan area network (MAN); a network of intermediate size, such as one spanning a local community.
    WAN: wide area network (WAN); links machines over a greater distance, perhaps in neigboring cities or on opposite sides of the world.
Network models
    Bus network: in which machines are connected to a common communication line.
    Star network: in which one machine serves as a central focal point to which all the others are connected.
Protocals
    CSMA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection (CSMA/CD); In Ethernet standards, this protocal dictates that each message be broadcasted to all the machines on the bus.
    CSMA/CA Wifi: Carrier Sense, Multiple Access with Collision Avoidance (CSMA/CA); standardized by Wifi or IEEE, wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them.
Combining Networks
    Repeater: litte more than a device that passes signals back and forth between the two original busses without considering the meaning of the signals.
    Bridge: similar to, but more complex than reapter. Looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a compouter on the others side.
    Switch: a bridge with multiple connections, allowing it to connect several buses rather than just two.
    Router: a device that allows connection bewteen networks to form an internet; special purpose computers used for forwarding messages.
    Gateway: serves as a passageway between the network and the outside world; the "point" at which one network is linked to an internet.
Client/ Server Model
    P2P: peer-to-peer (P2P); processes that provide service to and receive service from each other.
    Distributed system: software systems that consist of software units that execute as processes on different computers.
    Cloud computing: the latest trend in distributed systems, whereby huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing
    Tier 1: ISPs that consist of very high-speed, high-capacity, international WANs. Backbone of the internet.
    Tier 2: ISPs that tend to be more regional in scope and less potent in their capabilities. Distictiion between Teir 1 and 2 is often matter of opinion.
    Tier 3: an independent internet, sometimes called an intranet, operated by a single authority that is in the business of supplying Internet access to individual homes and businesses.
IP Address
    ICANN: Internet Corporation for Assigned Names and Numbers (ICANN)
    DNS: Domain Name System (DNS); name servers used as an Internet-wide directory systems, directories that provide address translation services to clients.
    TLD: Top Level Domain (TLD); used to reflect the domain's classification. Examples: .com - commercial industries, .gov - government institutions.
    Country-code TLDs: Click here to learn more.
Internet Protocols
    FTP: File Transfer Protocol (FTP); a protocoal for transmitting files between computers, an application for listing and copying files across the network.
    HTTP: Hyper Text Transfer Protocol (HTTP); an application protocal used to load webpages using hypertext links.
    SMTP: Simple Mail Transfer Protocol (SMTP); defines a way that two computers on the network may interact when transmitting an email message form one host to the other.
World Wide Web
    URL: Uniform Resource Locator (URL); a unique address that contains the information needed by a browser to contact the proper server and request the desired document.
    Client-side Activities: activities performed by client such as a browser.
    Server-side Activities: activities performed by server such as a webserver.
Internet four-layer software
    Application layer: Prepares message and provides destination address
    Transport layer: Chops message into packets, ensures that the messages are properly formatted for transmission over the internet.
    Network layer: Assigns intermediate address to each packet, decides which direction a packet should be sent at each step along the packet's path through the internet.
    Link layer: Transfers packet
Internet is:
    Packet-switching network: in which digital data is sent by breaking the data into blocks of bits called packets.
    fault tolerant: routing on the internet is fault tolerant and redundant.
    redundant: more than one way to route data.
Cybersecurity
    Malware: Malicious software, such software might be transferred to, and executed on, the computer itself, or it might attack the computer from a distance.
    Virus: a software that infects a computer by inserting itself into programs that already reside in the machine.
    Worm: an autonomous program that transfers itself through a netwrok, taking up residence in computers and forwarding copies of itself to other computers.
    Trojan hourse: a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victum.
    Spyware: another form of malicious software that collects information about activities at the computer on which is resides and reports that information back to the instigator of the attack.
    Phishing: a technique of obtaining information explicitly by simply asking for it.
    DDos attack: Distributed Denial of Service (DDoS); The work of generating the attacks is distributed over many hosts.
Protection
    Firewall: A primary prevention technique, filters traffic passing through a point in the network.
    Spam Filter: Firewalls designed to block unwanted emails.
    Proxy Server: A software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
    Antivirus Software : Used to detect and remove the presence of known viruses and other infections.
    Cryptography: Is the study of sending and receiving messages securely in the presence of adversaries.
    Public Key Encryption: Techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
