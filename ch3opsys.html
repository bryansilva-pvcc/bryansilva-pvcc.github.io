Operating Systems Terminology
TERM DEFINITION
operating system The software that contrls the overall operation of the computer including file management, the user interface, and the program execution environment.
job queue Programs, data, and directions residing in mass storage awaiting to be executud.
FIFO "first-in,first-out". A method for organizing data based on the item that is stored first is retrieved first. Most job queues don't follow the FIFO structure, most operating systems provide for concideration of job priorities.
interactive processing A feature that allowed a program being executed to carry on a dialogue with the user through remote terminals.
time-sharing Multiple users sharing access to a common computer.
multiprogramming A technique of implementing time-sharing, in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
application software programs for performing tasks particular to the machine's utilization. Examples include: spreadsheets, database systems, desktop publication systems, accounting system, and games.
system software provides the infrastructure that the appliction software requires. The interface between the hardware and user applications
utility software Programs designed for performing activities that are fundamental to computer installations but not included in the operating system. These activities aid the management and maintenance of a computer system. Activities include: disk management, backup and recovery, and system optimization.
graphical user interface (GUI) In which objects can be manipulated, such as files and programs, are represented pictorially on the display as icons. A user friendly way to navigate and manipulate files and programs.
shell An older user interface, communicated with users through textual messages using a keyboard and monitor screen. A command-line interface used to interact with operating system by typing commands.
kernel Internal part of an operating system. Performs the very basic functions required by the computer installation. Ensures proper communication between hardware and software components.
device driver Part of the kernal component, which are the software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine. Comunicates with objects such as printers, keyboards, and stoare drives.
file manager This units job is to coordinate the use of the machine's mass storage facilities. Allows users to access various files, and which portions of mass storage are available for new files or extensions to existing files.
memory manager Another part of the kernal component, which is charged with the task of coordinating the machines's use of main memory. Also enusres efficient use of availble memory.
virtual memory A large "fictional" memory space created by paging. It enables the memory manager to shuffle pages back and forth bwtween main memory and mass storage so that the pages that are needed at any given time are actually present.
boot strapping Performed by a computer each time it is turned on. It is this procedure that transfers the operating system from mass storage into main memory.
ROM "read-only memory". Contents that can be read but not altered. Memory that permanently stores firmware or system software.
deadlock The condition in which tow or more processes are blocked from progressing because each is waiting for a resouce that is allocated to another.
spooling A technique of holding data for output at a later but more convenient time.
super user (or administrator) This person gains highly privileged access to the operating systemby identifying him or herself as the administrator during the login procedure. The administrator can alter settings within the operating system, modify critical software packages, and adjust the privleges granted to other users.
multi-factor authentication Accounts that may require more than once piece of evidence to gain access. These factors can include passwords, a specialized security token, smartphone, and biometrics.
